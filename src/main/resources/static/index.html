Features

APGW is a web based platform for simplifying grading of coding assignments. It provides students a platform to code and submit programs and get it evaluated.

Open Source

APGW is licensed under GNU GPL v3. Anyone can use and modify it.

Automated Grading

Assignments are automatically graded, which means less hassle for teachers and students.

Cross Platform

APGW works on every device with a web browser.
